# Client Review Instructions

Use the **Client Review Template** sheet in `hot_tier_comparison.xlsx` (or the **Compare Export** from Settings).

## Columns to fill in

| Column | What to enter |
|--------|----------------|
| **Client Trusts Hot? (Y/N)** | `Y` if this Hot lead is someone you would prioritize for coaching outreach |
| **Client Wrong Hot? (Y/N)** | `Y` if this lead should **not** be Hot (wrong fit, book buyer, no intent, etc.) |
| **Client Notes** | Short note — e.g. “good parent, filled Wufoo”, “book only”, “already customer” |

## Suggested sample

- Mark **5–10 rows** you trust as Hot (`Client Trusts Hot? = Y`)
- Mark **5–10 rows** you think are wrong (`Client Wrong Hot? = Y`)
- Focus on **Dropped from Hot** and **Still Hot** sheets if time is limited

## Do not worry about

- Rows tagged **intentional_sparse_subscriber_cap** — these are email-only Subscribers; Cold is correct
- Small score changes (Hot → Warm) on leads with thin data — note them if outreach would still be worth it

## Return

Email the updated spreadsheet back. We use your marks to tune the model once — not to revert Subscriber protections.
